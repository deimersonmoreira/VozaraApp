# Branding - VozaraApp

## Conceito

**VozaraApp** é uma identidade simples para um app de transcrição local. A marca visual pode aparecer como **Vozara**, mais curta e memorável, enquanto o nome do projeto/repositório fica como **VozaraApp**.

Assinatura:

```text
Vozara
transcrição local de áudio e vídeo
```

Crédito:

```text
feito no ódio por @deimersonmoreira
instagram.com/deimersonmoreira
```

## Personalidade

- Clara

- Sem linguagem técnica pesada

## Tipografia

Use **Segoe UI** no Windows. É nativa, limpa e evita dependência externa.

Hierarquia sugerida:

- Título: Segoe UI Bold
- Botões: Segoe UI Semibold
- Texto comum: Segoe UI Regular

## Arquivos

- `assets/logo.svg` - logo principal com símbolo e assinatura.
- `assets/icon.svg` - ícone quadrado para atalho/instalador.
- `assets/wordmark.svg` - versão horizontal só com nome e assinatura.

## Observação jurídica

Esta identidade foi criada de forma autoral para o projeto, mas exclusividade comercial depende de pesquisa de marca e registro no INPI ou órgão equivalente.
