Dúvidas

Perguntas frequentes

O app precisa de internet para funcionar?
Só na primeira configuração, quando o assistente baixa o Python embutido, o FFmpeg e o modelo Whisper (~1,5 GB). Depois disso, tudo funciona completamente offline. Você pode até colocar o computador no modo avião e continuar transcrevendo.
Meus arquivos vão para algum servidor?
Não. O VozaraApp processa tudo localmente na sua máquina. Nenhum arquivo de áudio, vídeo ou texto transcrito é enviado para a internet. Isso é especialmente importante para profissionais que lidam com informações confidenciais de clientes.
A transcrição é em português?
Sim. O modelo está configurado especificamente para português brasileiro, com parâmetros otimizados para precisão máxima no idioma.
Posso usar no notebook sem GPU?
Sim. O app detecta automaticamente se há uma GPU NVIDIA disponível. Se não houver, ele usa a CPU - mais lento (10-30 min por hora de audio), mas funciona em qualquer computador Windows com 8 GB de RAM.
O que é "acesso gratuito vitalício"?
Durante a fase de testes, o VozaraApp é disponibilizado gratuitamente para membros. Quando o app for lançado oficialmente, o preço passará a ser R$19,90/mês. Quem baixar e se registrar durante a fase de testes nunca pagara nada - o acesso gratuito é permanente para esses membros, independente do preço futuro.
Quanto espaço o app ocupa?
Aproximadamente 8 GB no total: o modelo Whisper medium, o ambiente Python e o FFmpeg. As transcrições são arquivos de texto muito pequenos e ficam em Documentos\VozaraApp.
O app mexe nos meus arquivos originais?
Não. O VozaraApp nunca move, renomeia nem apaga seus arquivos de áudio e vídeo. Ele apenas lê o arquivo para gerar a transcrição e salva o resultado em pasta separada.
